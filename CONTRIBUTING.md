# Contributing

1. CMake ≥ 3.14 and a C11 compiler (GCC/Clang on Linux; MSVC or MinGW-w64 on Windows)
2. GUI: Python 3.8+ and tkinter on PATH - [docs/GUI.md](docs/GUI.md)
3. `cmake -B build && cmake --build build` → `build/overwrite` or `build/overwrite.exe`
4. Update `docs/` + `CHANGELOG.md` `[Unreleased]` for behavior changes

Security: [SECURITY.md](SECURITY.md)
